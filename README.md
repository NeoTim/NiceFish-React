# NiceFish-React

## 介绍
NiceFish的React版本。

## 软件架构

本项目采用React Hooks 16.8.3 版本，使用TypeScript、Ant Design组件库以及Bootstrap v4.2.1 开发

## 使用说明

打开你的命令行终端，执行以下命令：
- git clone https://gitee.com/mumu-osc/NiceFish-React.git 
- cd NiceFish-React 
- npm i 
- npm start 
打开你的浏览器，访问http://127.0.0.1:3001/
